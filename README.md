The application is using Next.js, Tailwind, Shadcn/ui and TypeScript. The unit testing is with Jest, the end-to-end testing is with Playwright.

First:

```bash
npm install
```

To run the development server:

```bash
npm run dev
```

To run units tests:

```bash
npm test
```

To run e2e test:

```bash
npm run build

npm run start

//in a different terminal
npx playwright test
```
